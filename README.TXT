PROJECT TITLE: Matching Picture Game
PURPOSE OF PROJECT: This can help a child's perceptual development
VERSION or DATE: 06/05/2016
AUTHORS: Amy Jin, Aya Keller, Michelle Zhou, Per. 4
USER INSTRUCTIONS:
Play by saying "true" or "false" if the picture given to the user (on the right) matches the "model" on the left. 
DO NOT CLICK ON THE BUTTONS ON THE GRID MORE THAN ONCE (after the picture is gone) BECAUSE YOU WILL MESS EVERYTHING UP.
